# fetch images
```
curl https://raw.githubusercontent.com/harryxinwang/images/master/scripts/fetch-kube-images.sh | bash
# push all images to docker hub
curl https://raw.githubusercontent.com/harryxinwang/images/master/scripts/fetch-kube-images.sh | bash -s push
# pull all images from docker hub
curl https://raw.githubusercontent.com/harryxinwang/images/master/scripts/fetch-kube-images.sh | bash -s pull
```
